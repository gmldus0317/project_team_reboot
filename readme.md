# Infornotice?
<h2>INformation FOR NOTICE</h2>
한국에서는 강배전 커피 등에 들어간 발암물질에 대한 정보가 많이 없다.
이 프로젝트는 흔히 마시는 커피 외에도 우리가 흔히 먹는 음식에 발암물질이 숨어 있다는 경각심을 주기 위해 진행되었다.
